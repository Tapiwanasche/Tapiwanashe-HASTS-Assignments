# Tapiwanashe HASTS Assignments

This repository contains my final Jupyter Notebook submissions for HASTS / Financial Econometrics coursework.

## Repository Contents

### Assignment 1: Best-Practices Handbook

**File:** `Tapiwanashe_J_Tapfumaneyi_Assignment_1.ipynb`

This notebook develops a best-practices handbook for selected financial time-series modeling challenges. The selected challenges include:

- Skewness
- Sensitivity to Outliers
- Overfitting
- Lack of Interpretation

Each challenge includes a technical definition, description, numerical demonstration, diagram, diagnosis, damage, and directions for improvement.

### Assignment 3: Application Time-Series Project

**File:** `Tapiwanashe_J_Tapfumaneyi_Assignment_3.ipynb`

This notebook applies a time-series model to financial market data. The selected model focuses on modeling non-stationarity and finding an equilibrium relationship between multiple financial time series.

The notebook includes:

- Model equations and term definitions
- Data import and preparation
- Parameter calibration
- Parameter interpretation
- Diagnostic plots
- Model limitations
- Deployment discussion

## Data Source

The notebooks use financial market data from Yahoo Finance through Python data-access tools.
